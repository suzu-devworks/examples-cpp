# Algorithm and Data Structure

## Environment

- Language: C11
- CMake 3.20 later
- or GNU Make 4.3

## Build

### Use CMake

```shell
# set clang
export CC=/usr/bin/clang
export CXX=/usr/bin/clang++

mkdir .build && cd .build
cmake ../

cmake --build .
```

### Use Make

```shell
make
```
